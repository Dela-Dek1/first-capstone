Student Tracker API Documentation

Base URL: /api/

Student Management

Endpoint Method Description
/api/students/ GET List all students
/api/students/ POST Create a new student
/api/students/{id}/ GET Retrieve a specific student
/api/students/{id}/ PUT/PATCH Update a student record
/api/students/{id}/ DELETE Delete a student

Attendance Tracking

Endpoint Method Description
/api/attendance/ GET List all attendance records
/api/attendance/ POST Create a new attendance record
/api/attendance/{id}/ GET Retrieve a specific attendance record
/api/attendance/{id}/ PUT/PATCH Update an attendance record
/api/attendance/{id}/ DELETE Delete an attendance record
/api/attendance/summary/ GET Get attendance statistics

Performance Tracking

Endpoint Method Description

/api/performance/summary/ GET Get performance statistics

Authentication

Note: All API endpoints require authentication.

You can authenticate using:

  • Session authentication (via login page)
  • Basic authentication (username/password in request header)

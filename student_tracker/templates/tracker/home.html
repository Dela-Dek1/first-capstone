The Student Tracker API provides the following endpoints:

  • /api/students/ - Student management
  • /api/attendance/ - Attendance tracking
  • /api/attendance/summary/ - Attendance statistics
  • /api/performance/ - Performance records
  • /api/performance/summary/ - Performance statistics
